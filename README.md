# Calgary Crime Dashboard (Part 4 – Prototype & Evaluation)

## Overview  
This project presents an interactive dashboard analyzing crime patterns in Calgary from 2018–2024. The dashboard allows users to explore how crime varies across communities, sectors, and time using both raw counts and normalized crime rates.

The dashboard is built using **D3.js** and focuses on interactivity, filtering, and clear visual comparisons.
